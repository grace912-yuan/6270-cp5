# Data Dictionary

## Dataset
CDC_vaccination_pregnancy_20250120.csv

## Variables
- Vaccine: vaccine type included in the dataset
- Geography Type: type of geography, such as states
- Geography: geographic location
- Survey Year/Influenza Season: survey year or influenza season
- Dimension Type: type of subgroup, such as race and ethnicity or age
- Dimension: subgroup category
- Estimate (%): estimated vaccination coverage percentage
- 95% CI (%): 95% confidence interval for the estimate
- Sample Size: number of respondents included in the estimate
